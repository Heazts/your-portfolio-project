Sobre Mim

Minha jornada no desenvolvimento começou com a paixão por entender como a web funciona. Desde então, tenho me dedicado ao estudo aprofundado de HTML, CSS, JavaScript e Python, construindo uma base sólida para desenvolver aplicações web.

Como desenvolvedor, estou sempre buscando aprimorar minhas habilidades. Sou curioso, proativo e gosto de enfrentar desafios. Acredito que um bom código é aquele que é limpo, eficiente e, acima de tudo, acessível.
